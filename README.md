Project: Cryptography Algorithms Implementation
Objective

Implement widely used cryptographic algorithms such as AES, RSA, and SHA to understand how encryption, decryption, hashing, and secure communication work. This project helps build a strong foundation in modern cryptography and cybersecurity.

Recommended Technology Stack
Programming Language
Python 3
Libraries
pycryptodome (modern replacement for PyCrypto)
hashlib (built into Python)
OpenSSL (for key generation and verification)
Development Tools
Visual Studio Code
Jupyter Notebook (optional)
Command Prompt/Terminal
